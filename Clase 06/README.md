# APUNTES

## ¿Qué es un objeto?
- Cualquier cosa que tenga una realidad aparente y una escencia. 
- El obejto tiene una composición: materila y conceptual; y relación. "Todo objeto está hecho para algo y sirve para algo".
- La escencia es algo que se encuentra en el objeto pero es incalcansable, existe pero no lo puedo conocer.
- Presentar los objetos tal como son: Eso "según el profe citando al autor" es arte.
- La afectación entre los objetos ocurre por medio de las cualidades accidentales de los objetos (peso, textura, temperatura, etc.). Las cualidades de los objetos varían con le tiempo.
- La opración refelxiva que es propia del arte: La metáfora. Es un ejerccio en donde puedo conectar dos objetos entre ellos, es decir,. combinar dos cualidades no escenciales entre dos elementos disciímiles. Así aparece un tercer un objeto: el objeto metafórico.
